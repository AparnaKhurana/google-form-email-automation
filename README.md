📩 Google Form Email Automation
🔹 Overview

This project is an automation workflow that connects Google Forms, Google Sheets, and Gmail to automatically send emails when a new form response is received. It eliminates manual work and ensures real-time communication.

🔹 How It Works
A user fills out a Google Form
Responses are stored in Google Sheets
A trigger detects a new row added
Gmail automatically sends an email to the user
🔹 Features
Real-time automation
Instant email notifications
No manual intervention
Simple and efficient workflow
